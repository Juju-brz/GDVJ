# Godot Video Jockey
 a real time generator of Pattern  and Shaders based on Godot 4.5

made by juju & Ali

This program use GDShaders for more information read the docs https://docs.godotengine.org/en/stable/tutorials/shaders/shader_reference/shading_language.html
if you are are good to GLSL for convert to Godot Shader https://docs.godotengine.org/en/stable/tutorials/shaders/converting_glsl_to_godot_shaders.html

F1 -> Reset
F2 -> show menu
F3 -> change mouse control for joyestick
F4 -> Change Scene
Alt + Enter -> Execute Shader Code
